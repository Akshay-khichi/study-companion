# Study Companion (Full-Stack Application)

An AI-powered full-stack learning platform that transforms PDFs and study topics into structured learning materials including notes, flashcards, quizzes, and an interactive AI tutor.

## Live Demo

https://study-companion-final.netlify.app

---

## Overview

Study Companion helps students convert lengthy study materials into concise, interactive learning resources. Users can upload lecture PDFs or enter a topic, and the platform automatically generates notes, flashcards, quizzes, and AI-powered tutoring assistance.

The platform is designed to improve learning efficiency, retention, and understanding by transforming passive study material into an active learning experience.
